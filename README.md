# 📝 To-Do List Web Application

A simple and user-friendly To-Do List web application built using HTML, CSS, and JavaScript. This application allows users to create, manage, complete, and delete tasks efficiently.

---

# 🚀 Features

✅ Add New Tasks  
✅ Mark Tasks as Completed  
✅ Delete Tasks  
✅ Responsive User Interface  
✅ Local Storage Support  
✅ Dynamic Task Management  

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

# 📋 Functionalities

## ➕ Add Tasks
Users can add tasks using the input field.

## ✅ Complete Tasks
Users can mark tasks as completed.

## ❌ Delete Tasks
Users can remove unwanted tasks.

## 💾 Save Tasks
Tasks are automatically saved in browser local storage.
